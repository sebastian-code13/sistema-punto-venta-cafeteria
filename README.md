# sistema-punto-venta-cafeteria
Sistema de escritorio desarrollado en Java para la gestión de una cafetería, incluyendo control de ventas, clientes e inventario.

Tecnologías utilizadas:
Java
MySQL
NetBeans

Funcionalidades:
Gestión de productos
Registro de ventas
Control de inventario
Sistema de clientes con puntos
Roles de usuario (cajero / gerente)
Generación de tickets
